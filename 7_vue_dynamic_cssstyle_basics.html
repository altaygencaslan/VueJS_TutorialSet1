Dynamic CSS Basics

Dinamik CSS kullanımında :class="{ key : value}" şeklinde değer alır. Burada "key" atanmasını istediğimiz değer. Aşağıdaki örnek için bu yellow. "value" ise ne olduğunda veya hangi şart sağlandığında bu class'ın (yellow) ekleneceği şartıdır. Hızlı bir testamacıyla aşağıda key bilgisine yellow ve value bilgisine de true yazarsak kutu sarı olacaktır, false yazarsak kutu gri kalacaktır. Aşağıdaki örnekte 2 adet class görülmekte ve bunlar çakışmamaktadır zira göre soldaki standart class bilgisi element ile gelebilir ancak sağdaki Vue ile bind edilmiş class ise Vue ile alınıp render edilen class bilgisidir ve Vue bu ayrımı yapabilir. Ayrıca key-value çiftleri seri halinde birkaç adeteklenebilir.
